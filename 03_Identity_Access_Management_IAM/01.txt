* IAM 101

# What is IAM
IAM allows you to manage users and their level of access to the AWS console. It is important to understand IAM and how it works, both for the exam and for administrating a company's AWS account in real life.

# What does IAM give you
1. Centralized control of your AWS account.
2. Shared access to your AWS account.
3. Granular Permissions
4. Identity Federation (including Active Directory, Facebook, LinkedIn etc)
5. Multifactor Authentication
6. Provide temporary access for users/devices and services where necessary.
7. Allows you to set up your own password rotation policy.
8. Integrates with many different AWS services.
9. Supports PCI DSS compliance.

# Critical Terms
1. Users - End users (think people)
2. Groups - A collection of users under one set of permissions.
3. Roles - You create roles and can then assign them to AWS resources.
4. Policies - A documents that defines one or more permissions.
